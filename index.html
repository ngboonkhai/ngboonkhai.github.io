Developer Tools

Welcome to ngboonkhai's Developer Tools

A collection of free, lightweight, browser-based utilities built for engineers, students, and open-source contributors. No installs, no sign-ups — just open and use.

🧮

32-Bit Manipulation Tool

Visually edit individual bits of a 32-bit register. Rotate, shift, invert, and zero bits with one click. Instantly see the resulting hexadecimal and decimal values. Ideal for embedded systems engineers and anyone working with hardware registers or bitwise operations.

  • Click any bit to toggle it
  • Rotate left / right, shift left / right
  • 1's complement & zero all bits
  • Fill from hex or decimal input
✉️

Email Quote Appender

Automatically prepends > to every line of an email, following the plain-text quoting convention used by the U-Boot and Linux kernel mailing lists. Supports nested quoting, configurable 80-column line wrapping, and one-click copy.

  • Live output as you type or paste
  • Correctly handles already-quoted lines
  • Auto word-wrap at configurable column limit
  • One-click copy to clipboard

About These Tools

All tools run entirely in your browser — no data is sent to any server. They are open-source (MIT licence) and hosted on GitHub Pages. Use the sidebar on the left to switch between tools, or click the buttons above to jump straight in.

Who Are These For?

– Embedded software and firmware engineers debugging hardware registers.
– Linux kernel and U-Boot contributors replying to patch review emails.
– Students learning computer architecture, bitwise operations, or open-source contribution workflows.
