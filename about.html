👤 About

About

Hi, I'm ngboonkhai — an embedded software engineer with a passion for building small, practical tools that solve real everyday problems faced by engineers and developers.

Why This Site?

Many day-to-day tasks in embedded development and open-source contribution involve repetitive steps — toggling bits in a register, formatting an email reply for a mailing list — that don't have a convenient dedicated tool. This site collects those small utilities in one place, free and ad-light.

The Tools

  • 32-Bit Manipulation Tool — a visual bit editor for 32-bit hardware registers. Useful for embedded engineers, students learning computer architecture, and anyone working with bitwise operations.
  • Email Quote Appender — automatically prepends > to every line for plain-text email replies, following the convention used by the U-Boot and Linux kernel mailing lists.

Open Source

All tools are open source under the MIT licence and hosted on GitHub:

  • web.bit-manipulation-tool
  • web.email-quote-appender
  • ngboonkhai.github.io (this site)
